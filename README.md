# basic-blockchain
Basic cryptocurrency, based on blockchain, implemented in Python

## Getting Started

### Installation
```
pip install bbchain
```

## Development
```bash
git clone https://github.com/jepemo/basic-blockchain.git
cd basic-blockchain
python3 -m venv venv
source venv/bin/activate
pip install -e .

bbchain --help
```
